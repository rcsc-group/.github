# Papers with Code

A chronological index of publications from the [Radu Cimpeanu Scientific Computing Group](https://www.raducimpeanu.com) accompanied by open-source code and data repositories.

All simulations are reproducible from the linked repositories. If you encounter any issues running the code, please open an issue in the relevant repository or just let us know.
